# HolyGround Next.js Demo

Production-style Next.js fullstack prototype built for a Catholic theology platform.

## 🔴 Live Demo

https://holyground-demo.vercel.app/

## ✨ Features

### 📖 Online Bible Reader

- Dynamic book & chapter navigation
- Clean scripture reading interface
- Server-side API fetching
- Optimized for readability and performance

### 🔎 Scripture Search

- Search verses instantly (example: John 3:16)
- Fast API-based search
- Structured verse rendering

### 🎁 Lead Magnet Signup Funnel

- Conversion-focused popup modal
- Email capture with validation
- API route for signup handling
- Simulated automated eBook delivery flow
- Designed to mirror real marketing funnel behavior

## ⚙️ Tech Stack

- Next.js 16 (App Router + Fullstack API routes)
- TypeScript
- Tailwind CSS
- Vercel deployment
- Accessibility-friendly form components

## 🧠 Purpose

This project was built as a working prototype to demonstrate how a modern Catholic theology website could implement:

- A lead magnet funnel for conversion
- An online Bible reader experience
- A scalable search feature
- Clean and maintainable Next.js architecture
